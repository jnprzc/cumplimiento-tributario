# Frontend - Sistema de Autodiagnóstico Tributario

## Descripción
Interfaz web para consultar el estado de cumplimiento tributario de empresas colombianas.

## Archivos
- `index.html` - Estructura de la página
- `styles.css` - Estilos y diseño
- `script.js` - Lógica y conexión con API
- `config.js` - Configuración de la aplicación

## Uso en Desarrollo

1. Asegúrate de que la API esté corriendo en `http://localhost:8000`
2. Abre `index.html` en tu navegador

## Características

- ✅ Diseño responsivo (móvil y desktop)
- ✅ Validación de NITs en tiempo real
- ✅ Animación del score de cumplimiento
- ✅ Captura de leads por email
- ✅ Manejo de errores
- ✅ Scroll automático a resultados
